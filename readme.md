## Como rodar a aplicação

### Instalando dependencias

Na raiz do projeto rode:

```
pip install -r ./requirements.txt
```

### Subindo a aplicação

Rode na raiz do projeto
```
fastapi dev main.py
```

## Acessando a api

localhost:8000


```
docker build -t tic-tac-toe-image .
docker run -d -p 8000:8000 tic-tac-toe
```
